## 📚 Учебные задания

### 1. DOM-тест «Ругательства» (`index.html`)

Интерактивный тест на знание устаревших русских ругательств:

- 4 вопроса с вариантами ответов
- После выбора – объяснение правильного ответа
- Анимация правильного / неправильного выбора
- Подсчёт результатов
- Возможность просмотреть все вопросы с ответами

### 2. CSS-рисунок (`картинка.html`)

Полностью CSS-фигурка:

- 🐴 Осёл (тело, голова, уши, ноги, хвост)
- 🎈 Воздушный шар с ниткой
- 🌱 Трава с градиентом

### 3. Flexbox-анимация (`анимация.html`)

Анимация букв **A** и **B**:

- при наведении – движение по кривой
- при фокусе – прыжки + появление изображения
- в конце – эффект сепии и текст «КОНЕЦ»
